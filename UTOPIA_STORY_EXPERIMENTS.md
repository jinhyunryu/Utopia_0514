# UTOPIA_STORY_EXPERIMENTS
# 스토리 접목 실험 기록

> 이 문서는 원본 설정이 아니다.  
> 여기에 적힌 내용은 확정(LOCK)이 아니며, 사용자의 명시 승인 전까지 `UTOPIA_CORE.md`, `UTOPIA_DATA.md`, `UTOPIA_DESIGN.md`, `UTOPIA_APPENDIX.md`에 반영하지 않는다.  
> 목적은 “어울릴 수도 있는 스토리 수정안”을 실제 원본 수정 전에 안전하게 시험·비교·폐기하기 위함이다.

---

## 0) 상태 태그

- **EXPERIMENT**: 접목 실험 중
- **FIT_CHECK**: 기존 설정과 어울리는지 검토 중
- **KEEP_CANDIDATE**: 유지 후보
- **NEED_REWORK**: 수정 후 재검토
- **REJECTED**: 폐기
- **APPROVED**: 사용자 승인 완료
- **APPLIED**: 원본 문서 반영 완료

---

## 1) 사용 원칙

- 이 파일의 내용은 LOCK이 아니다.
- 기존 원본과 충돌하면 원본을 우선한다.
- Codex는 이 파일의 내용을 근거로 원본 문서를 자동 수정하지 않는다.
- 스토리 접목안은 반드시 “어울리는 이유 / 어색한 이유 / 충돌 가능성”을 함께 기록한다.
- 사용자가 승인하기 전에는 `UTOPIA_DECISIONS.md`로 옮기지 않는다.
- 원본에 실제 반영된 뒤에만 `CHANGELOG.md`에 기록한다.
