#### D3.1.a) 직업 보정치 실제 수치 체계 (LOCK)

> 직업 보정치는 각 빙의체의 전투 숙련도, 현장 대응력, 무장 활용도, 역할 난이도를 반영하는 수치다.  
> 종족 기본 체급 위에 더해져 초기 체급 지표(HP + 속도)의 방향성을 결정한다.

- **최상위 전면 전투원**: **+125 ~ +130**
  - 전면 교전 능력, 생존력, 압박력이 모두 높은 직업군
  - 예: Lysa(+125), Korin(+125), Mokhra(+130)

- **돌파 / 기동 전투원**: **+105 ~ +115**
  - 돌입, 추적, 선봉, 고위험 교전 능력이 높은 직업군
  - 예: Caldus(+110), Zugrim(+115), Nyra(+105)

- **숙련 전술가 / 정밀 전투원**: **+95 ~ +100**
  - 교전 능력은 충분하지만, 전면 압살보다 정밀 운용과 역할 수행에 강한 직업군
  - 예: Eirwen(+95), Brena(+100), Vaelis(+100)

- **비전 / 학자 / 예언형**: **+90**
  - 직접적인 전면 교전 능력보다 해석, 제어, 비전 운용, 특수 역할 비중이 높은 직업군
  - 예: Maelor(+90), Rhexar(+90), Rhyden(+90)

### D3.2) 캐릭터 전투 스탯 종류 (LOCK)

- **HP**: 생명력. 0이 되면 사망한다.
- **속도**: 타임라인 순환 속도에 영향을 주는 기본 스탯이다.
- **이동력**: 자신의 턴에 기본적으로 이동 가능한 칸 수다.
- **공격력**: 시엔 및 베라 스킬의 피해 계산에 사용되는 기본 스탯이다.
- **방어력**: 적에게서 받는 피해를 경감하는 기본 스탯이다.
- **브레이크 저항**: 자신이 받는 브레이크 게이지 피해를 줄여 주는 스탯이다.
- **상태이상 저항**: 독, 이동 불가, 약점 노출, 공격력 약화 등 각종 상태이상 및 디버프에 대한 저항 능력이다.

## D7) 공용 베라(Vera) 시스템

### D7.1) 베라 운용 구조
- 베라는 요한이 사용하는 공용 플레이어 스킬 시스템이다.
- 모든 빙의체는 동일한 베라 시스템을 공유한다.
- 베라 스킬 계통은 단일 / 광역 / 생존 / 버프로 구성된다.
- 단일 계통은 초기 해금 상태다.
- 광역 / 생존 / 버프 계통은 Soul UI에서 SP를 투자해 순차 해금한다.
- 플레이어는 해금된 베라 계통 중 현재 전투에 사용할 베라 스킬 1개를 선택해 운용한다.
- 베라 게이지는 턴당 1씩 충전되며 최대치는 10이다.
- 베라 게이지는 전투 간 누적 이월된다.
- 라그니스 던전에서 잔핵을 회수하면 현재 사용 중인 베라 스킬이 해당 회차에 한해 강화된다.

  #### D9.x) 룬 합성 및 등급 상승 로직 (LOCK)

- 같은 등급의 룬 3개를 합성하면 확률에 따라 상위 등급 룬 1개를 획득한다.
- 합성 결과는 기본적으로 다음 등급이 우세하며, 낮은 확률로 그보다 더 높은 등급이 출현할 수 있다.

- **노말 3개 합성**: 언커먼 92% / 레어 7% / 유니크 1%
- **언커먼 3개 합성**: 레어 92% / 유니크 7% / 레전드 1%
- **레어 3개 합성**: 유니크 94% / 레전드 6%
- **유니크 3개 합성**: 레전드 100%

- 레전드는 최종 등급으로, 추가 합성 대상이 아니다.
- 특수 조합 합성으로 완성되는 **Relic 룬**은 위 확률 합성과 별도의 조합 규칙을 따른다.
- Relic 룬은 일반 합성 확률 결과로는 등장하지 않는다.

### D9.3) 룬 이름 표기 규칙 (개정 LOCK)
- 일반 룬의 이름은 **각인**과 **인장**의 두 계열로 통일한다.
- **각인 (Brand)**은 공격, 조건부 강화, 물리 타입 강화, 브레이크, 생존 반응 등 전투 성능 강화 계열에 사용한다.
- **인장 (Seal)**은 속성 부여, 상위 속성, 초월, 범위 변화, 구조 변형, 장판, 상태이상, 특수 메커니즘 계열에 사용한다.
- 특수 조합 합성으로 완성되는 **Relic 룬**은 일반 룬과 구분되는 조합 전용 결과물로, 한글은 **~문**, 영어는 **Phrase of ~** 형식을 기본으로 사용한다.
- 같은 계열의 룬은 가능한 한 동일한 접미 체계를 유지한다.

### D9.4) 룬 장착 체계 (LOCK)
- 모든 빙의체는 시엔 6개를 가지며, 각 시엔 스킬마다 룬 소켓 2개를 가진다.
- 총 장착 가능한 룬 수는 **12개**다.
- 룬은 빈 소켓에 즉시 장착할 수 있다.
- 룬은 특정 캐릭터 전용이 아니라 **범용 장착**을 전제로 설계한다.

### D9.5) 룬 분류 체계 (개정 LOCK)
- **범용 룬**:
  - 대부분의 시엔 스킬에 장착 가능한 기본 강화 룬
  - 피해량, 사거리, 브레이크, 실드, 타임라인 지연 등 보편 효과를 다룬다.
- **조건부 룬**:
  - 특정 조건을 만족할 때만 효과가 발동되는 룬
  - 약점 적중, 표식 대상, 독 상태 대상, 브레이크 상태 대상, 물리 타입, 버프 상태 등을 조건으로 사용한다.
- **특수 메커니즘 룬**:
  - 스킬의 작동 방식 자체를 바꾸는 룬
  - 단일 → 관통, 장판 생성, 중심 당김, 타격 후 이동, 재타격, 위치 교환 등 구조 변형을 담당한다.
- **원소 부여 룬**:
  - 무속성 또는 기본 원소 스킬에 물 / 불 / 바람 속성을 부여하는 룬
  - 레전드 등급에서는 빛 / 어둠 상위 속성, 초월 특수 룬까지 포함한다.
- **상태이상 강화 룬**:
  - 독, 이동 불가, 약점 노출, 공격력 약화, 속도 지연 같은 상태이상/디버프 효과를 강화하는 룬
  - 독은 속성이 아니라 이 분류에서 다룬다.
- **물리 타입 연계 룬**:
  - 참격 / 관통 / 타격 스킬 전용으로 작동하는 룬
  - 물리 타입 정체성을 강화한다.
- **버프 / 자기강화 룬**:
  - 자기 버프, 실드, 방어 태세, 은신, 강화 상태의 성능을 높이는 룬
- **생존 / 회복 룬**:
  - 실드, 조건부 회복, 전투 종료 후 회복, 저체력 역전 같은 유지력 효과를 다루는 룬
- **특수 조합 룬 (Relic)**:
  - 지정된 조합식으로만 획득하는 상위 결과물
  - 일반 룬보다 빌드 방향이 더 선명하며, 용언 구조를 부분 복원한 결과물로 취급한다.

### D9.6) 속성 부여 룬 규칙 (LOCK)
- 플레이어 캐릭터의 기본 스킬은 물 / 불 / 바람만 직접 사용한다.
- **빛 / 어둠은 룬을 통해 부여되는 상위 속성**이다.
- **초월은 레전드급 예외 효과 또는 특수 룬으로만 부여되는 최상위 속성**이다.
- 속성 부여 룬이 장착된 스킬은 기존 기본 속성 대신 부여된 속성으로 판정한다.
- **독은 속성이 아니므로 속성 부여 룬으로 다루지 않는다.**
- 독은 별도의 상태이상 강화 룬으로만 다룬다.

### D9.7) 룬 장착 제한 규칙 (LOCK)

- 속성 부여 룬은 각 시엔 스킬당 1개만 적용할 수 있다.
- 빛 / 어둠 / 초월 계열은 동시에 중첩 적용할 수 없다.

### D9.18) 특수 조합 합성 룬 규칙 (LOCK)
- 일부 룬은 일반 합성 결과로 등장하지 않고, **지정된 룬 3개의 조합식**을 통해서만 획득할 수 있다.
- 특수 조합 합성 룬의 결과 등급은 전부 **Relic**으로 통일한다.
- **Relic 룬은 일반 드랍 룬 등급과 별도의 조합 전용 등급**이다.
- Relic 룬은 일반 합성 확률 결과로는 등장하지 않는다.
- Relic 룬은 지정된 조합식으로만 획득할 수 있다.
- 특수 조합 합성은 재료 룬 3개를 소모해 결과 룬 1개를 확정 획득한다.
- 특수 조합 합성은 반드시 같은 등급 3개만 요구하지 않으며, **서로 다른 등급의 재료 룬 조합**을 사용할 수 있다.
- 결과 룬 등급은 재료 룬의 평균 등급으로 계산하지 않으며, 조합식에 의해 고정된다.

## D10) 속성 시스템 데이터 (개정 LOCK)
> 연관 섹션: C4.x, D7.7

### D10.1) 속성 운용 규칙
> 연관 섹션: C4.x.1, C4.x.2, D7.7

- 기본 원소 속성은 **물 / 불 / 바람**으로 구성한다.
- 상위 원소 속성은 **빛 / 어둠**으로 구성한다.
- 최상위 속성은 **초월**이다.
- 기본 원소 상성은 **물 > 불 > 바람 > 물**의 순환 구조를 따른다.
- 상위 원소 상성은 **빛 ↔ 어둠**의 상극 구조를 따른다.
- 초월 속성은 기본 원소 및 상위 원소 상성과 별도로 작동하는 최상위 속성이다.
- 플레이어 캐릭터는 기본적으로 **물 / 불 / 바람** 계열 스킬만 직접 사용한다.
- **빛 / 어둠은 룬을 통해 기존 스킬에 부여되는 상위 속성**으로 운용한다.
- **초월은 최상위 룬 또는 특수 효과를 통해서만 부여되는 예외적 속성**으로 운용한다.
- 상위 속성 또는 최상위 속성이 부여된 스킬은 기존 기본 속성 대신 **부여된 속성으로 판정**한다.
- 물리 스킬은 무속성이며, 기본적으로 원소 약점 / 저항 판정에 포함되지 않는다.
- 속성의 개성은 고정 상태이상이 아니라 **스킬별 고유 효과**로 표현한다.

### D10.2) 독 상태이상 상세 규칙
> 연관 섹션: C4.x, D7.7

- **독은 속성이 아니라 상태이상**이다.
- 독은 특정 스킬, 룬, 적 효과에 의해 부여된다.
- 독은 최대 3중첩까지 누적된다.
- 1중첩: 대상 최대 HP의 2% 고정 피해
- 2중첩: 대상 최대 HP의 5% 고정 피해
- 3중첩: 대상 최대 HP의 8% 고정 피해
- 독 지속시간은 기본 3턴이며, 새 중첩이 쌓이면 지속시간이 갱신된다.

### D10.3) 초월 상세 규칙 (개정 LOCK)
> 연관 섹션: C4.x.3, D7.7

- 초월 속성은 **최상위 속성**이다.
- 초월 속성이 부여된 공격은 대상의 기존 약점 여부와 관계없이 **약점 공격 판정**을 적용한다.
- 초월 속성이 부여된 공격은 **속성 저항의 영향을 받지 않는다**.
- 초월 속성이 부여된 공격은 **브레이크 누적이 50% 증가한다**.
- 초월 속성은 기본 원소 및 상위 원소 상성을 따르지 않는 예외적 속성이다.

### D11.0) 몬스터 AP 및 스킬 운용 규칙 (재개정 LOCK)

#### D11.0.1) 몬스터 AP 및 이동 운용 규칙

- 몬스터도 플레이어와 동일한 AP Banking 규칙을 따른다.
- 몬스터의 일반 이동은 AP가 아니라 **이동 코스트**를 사용한다.
- 몬스터도 자신의 턴 시작 시 **이동 코스트 3**을 획득한다.
- 몬스터의 이동은 기본적으로 **1칸당 이동 코스트 1**을 소모한다.
- 몬스터 스킬도 플레이어의 시엔 스킬과 마찬가지로 AP를 소모해 사용한다.
- 몬스터는 역할군과 개체 등급에 따라 시작 AP, 최대 AP, 턴 회복량에 차등을 둔다.
- 몬스터의 이동 코스트는 턴 종료 시 소멸하며, 다음 턴으로 이월되지 않는다.
- 돌진, 도약, 순간이동, 위치 교환 같은 스킬 이동은 일반 이동과 별도로 처리한다.

#### D11.0.2) 몬스터 등급별 AP 기준

- **일반 몬스터**: 시작 AP **6** / 최대 AP **10** / 턴 회복 **4**
- **엘리트 몬스터**: 시작 AP **10** / 최대 AP **15** / 턴 회복 **5**
- **보스 몬스터**: 시작 AP **10** / 최대 AP **15** / 턴 회복 **5**

- 일반 몬스터는 다수 출현을 전제로 하므로, 플레이어와 동일한 AP 수치를 사용하지 않는다.
- 일반 몬스터의 위협은 개체당 폭발력보다 숫자, 포지션, 역할 분담에서 나온다.
- 엘리트와 보스는 플레이어와 같은 기본 AP 체계를 사용하되, 스킬 구조와 기믹으로 위협을 설계한다.
- **군집형 특수 적**은 예외적으로 개체별 AP 대신 별도의 군집 AP 규칙을 사용할 수 있다.

#### D11.0.3) 몬스터 스킬 구성 규칙

- 몬스터 스킬은 쿨타임보다 AP 소모를 중심으로 운용한다.
- **일반 몬스터**는 2~3개의 스킬을 가진다.
- **엘리트 몬스터**는 3~4개의 스킬을 가진다.
- **보스 몬스터**는 4~6개의 스킬을 가진다.
- 몬스터 스킬은 플레이어 시엔 스킬과 같은 전장 규칙을 따르며, 사거리 / 범위 / 넉백 / 추락 / 브레이크 / 상태이상 / 장판 효과를 사용할 수 있다.
- 몬스터는 역할군에 따라 스킬 우선순위가 다르며, 모든 스킬을 균등 확률로 사용하지 않는다.

#### D11.0.4) 몬스터 스킬 기본 틀

- **일반 몬스터**
  - 기본 공격기 1
  - 역할 특화기 1
  - 선택 행동기 1

- **엘리트 몬스터**
  - 기본 공격기 1
  - 역할 특화기 1
  - 제어 또는 이동기 1
  - 고위협 스킬 1

- **보스 몬스터**
  - 기본 공격기 1
  - 핵심 패턴 스킬 2
  - 제어기 1
  - 전장 기믹 연계기 1
  - 결전기 1

#### D11.0.5) 몬스터 스킬 AP 코스트 가이드

- **소형기**: 2~3 AP
- **중형기**: 4~5 AP
- **대형기**: 6~8 AP

- 일반 몬스터는 주로 소형기와 중형기를 사용한다.
- 일반 몬스터는 기본적으로 **대형기를 사용하지 않는다.**
- 엘리트 몬스터는 중형기와 대형기를 혼합 사용한다.
- 보스 몬스터는 전용 패턴에 따라 대형기를 사용하되, 과도한 연속 폭딜보다 예측 가능한 압박 구조를 우선한다.

#### D11.0.6) 몬스터 행동 우선순위 규칙

- 몬스터는 항상 현재 AP로 가능한 최적 행동만 찾는 것이 아니라, 역할군에 따른 우선순위를 가진다.
- **돌격형**은 거리 축소와 직접 공격을 우선한다.
- **추적형**은 위치 확보와 도주 차단을 우선한다.
- **원거리 견제형**은 사거리 유지와 안전한 공격을 우선한다.
- **지형 활용형**은 벽꽝, 추락, 함정 연계를 우선한다.
- **방어형**은 길막, 보호, 시간 끌기를 우선한다.
- **엘리트와 보스**는 한 턴에 공격과 제어를 결합한 복합 행동을 우선적으로 시도한다.

#### D11.0.8) 몬스터 전장 활용 AI 규칙 (LOCK)

- 몬스터는 단순히 가장 가까운 대상을 향해 돌진하지 않고, **역할군과 전장 상태에 따라 우선순위 행동**을 선택한다.
- 몬스터 AI는 완전 정보형 퍼즐이 아니라, **읽을 수 있지만 무시하면 아픈 규칙형 AI**를 지향한다.
- 몬스터는 전장 오브젝트, 엄폐, 협로, 고저차, 브레이크 상태를 인식하고 행동에 반영한다.

- 몬스터는 아래 우선순위에 따라 행동을 선택한다.
  1. **즉시 처치 또는 고위협 공격 가능 여부 확인**
  2. **브레이크 유도 또는 브레이크 대상 마무리 가능 여부 확인**
  3. **전장 오브젝트 / 기믹 활용 가능 여부 확인**
  4. **엄폐 확보 또는 플레이어 엄폐 차단 가능 여부 확인**
  5. **유리한 거리 확보**
  6. **기본 공격 또는 역할 수행**

- 몬스터는 항상 최적해만 고르지 않지만, 자신의 역할군에 맞는 행동을 우선한다.
- 몬스터는 플레이어가 예상 가능한 범위 안에서 전술적으로 행동해야 한다.

##### 역할군별 AI 성향 (LOCK)

- **돌격형**
  - 플레이어와의 거리 축소를 최우선한다.
  - 협로가 열려 있으면 정면 돌입을 우선한다.
  - 넉백, 벽꽝, 추락 유도가 가능하면 이를 우선 사용한다.

- **방어형**
  - 좁은 통로, 문, 엄폐 인접 칸을 선점한다.
  - 플레이어의 접근 경로를 막는 행동을 우선한다.
  - 직접 처치보다 시간 끌기와 공간 통제를 우선한다.

- **원거리 견제형**
  - 완전 엄폐가 없는 사선을 확보하려 한다.
  - 자신의 앞에 탱커나 장애물이 있을 경우 후열 유지 행동을 우선한다.
  - 플레이어가 반엄폐에 숨으면 위치를 옮기거나 범위기로 압박한다.

- **추적형**
  - 도주 중인 플레이어, 약한 체력의 대상, 표식 대상 추격을 우선한다.
  - 단순 최단거리보다 끼어들기, 측면 진입, 퇴로 차단을 선호한다.

- **제어형**
  - 이동 불가, 타임라인 지연, 위치 교란, 오브젝트 연계를 우선한다.
  - 즉시 큰 피해보다 다음 턴의 행동 불리함을 만드는 쪽을 선호한다.

- **기믹 연계형**
  - 전장 오브젝트 근처에서 싸우며, 플레이어를 그쪽으로 유도한다.
  - 추락, 벽꽝, 폭발, 도관, 해머 같은 전장 요소를 직접 활용한다.
  - 해당 전장에선 일반 공격보다 기믹 연계를 우선한다.

##### 엄폐 활용 규칙 (LOCK)

- 원거리 몬스터는 가능한 경우 **반엄폐 또는 완전 엄폐 인접 위치**를 우선 점유한다.
- 플레이어가 완전 엄폐 뒤에 있을 경우, 원거리 몬스터는 다음 중 하나를 선택한다.
  - 측면 이동으로 사선 확보
  - 범위기 사용
  - 노출된 대상 우선 공격
  - 기믹형 오브젝트 작동
- 근접 몬스터는 플레이어가 엄폐 뒤에 있더라도, 우회 경로가 짧으면 돌입을 시도한다.

##### 오브젝트 활용 규칙 (LOCK)

- 몬스터는 전장 오브젝트의 기능을 인식한다.
- 오브젝트 활용 가능 조건이 충족되면, 일반 공격보다 높은 우선순위로 사용할 수 있다.
- 몬스터는 다음 상황에서 오브젝트 활용을 우선한다.
  - 플레이어를 추락/벽꽝 위치로 유도 가능할 때
  - 오브젝트 발동으로 다음 플레이어 턴을 약화시킬 수 있을 때
  - 협로 차단, 문 닫힘, 사슬 당김 등으로 공간 이득을 얻을 수 있을 때
- 엘리트와 보스는 일반 몬스터보다 오브젝트 활용 빈도가 높다.

##### 타임라인 인식 규칙 (LOCK)

- 몬스터는 플레이어와 자신의 타임라인 상태를 인식한다.
- 브레이크 직전의 플레이어, 다음 턴이 빠른 플레이어, 위험한 연속 행동 가능 플레이어를 우선 압박할 수 있다.
- 제어형 몬스터는 타임라인 지연으로 플레이어 턴을 끊는 행동을 우선 선택할 수 있다.
- 브레이크 상태 대상이 있으면, 해당 대상을 마무리하거나 추가 압박하는 행동의 우선순위가 높아진다.

##### 브레이크 대응 규칙 (LOCK)

- 몬스터는 자신이 브레이크 직전일 경우, 가능하면 다음 행동을 우선한다.
  - 플레이어를 밀어내거나 당겨서 거리 재조정
  - 엄폐 뒤로 이동
  - 브레이크 유발 가능성이 높은 대상을 공격
  - 방어 버프 또는 피해 감소 행동
- 엘리트와 보스는 일반 몬스터보다 브레이크 회피 행동 성향이 강하다.
- 일반 몬스터는 브레이크를 완벽히 피하지 못해도 괜찮지만, 엘리트 이상은 어느 정도 저항해야 한다.

#### D11.0.9) 엘리트 / 보스 AI 강화 규칙 (LOCK)

- 엘리트와 보스는 일반 몬스터보다 전장, 오브젝트, 타임라인을 적극적으로 활용한다.
- 이들은 단순히 수치가 높은 적이 아니라, **전술 규칙을 더 깊게 사용하는 적**으로 설계한다.

##### 엘리트 AI 강화 규칙 (LOCK)
- 엘리트는 자신의 역할군 행동 우선순위를 더 강하게 따른다.
- 엘리트는 브레이크 직전일 때 생존 행동 또는 거리 재조정을 더 자주 선택한다.
- 엘리트는 전장 오브젝트를 일반 몬스터보다 우선적으로 활용한다.
- 엘리트는 플레이어의 엄폐, 협로, 고저차 이점을 무력화하려는 행동을 시도한다.
- 엘리트는 플레이어의 약한 체력, 브레이크 직전 상태, 고립된 위치를 우선 압박한다.
- 엘리트는 한 턴에 공격과 제어를 결합한 행동을 할 수 있도록 스킬 구성을 가진다.

##### 보스 AI 강화 규칙 (LOCK)
- 보스는 전장을 자신의 패턴 일부로 사용한다.
- 보스는 오브젝트, 기믹, 협로, 추락 지점, 차단 구역을 단순 참고가 아니라 **패턴 연계 요소**로 활용한다.
- 보스는 특정 전장 요소와 연계될 때 더 강한 행동을 우선한다.
- 보스는 플레이어가 익힌 안전지대를 오래 유지하지 못하도록, 위치 변경 또는 전장 상태 변화를 유도한다.
- 보스는 브레이크를 당하기 전과 후의 행동 우선순위가 다를 수 있다.
- 보스는 브레이크를 회피하려 하거나, 브레이크 후 회복 구간을 최소화하려는 패턴을 가질 수 있다.

##### 엘리트 / 보스의 타임라인 활용 규칙 (LOCK)
- 엘리트와 보스는 타임라인이 빠른 플레이어를 더 위협적인 대상으로 인식할 수 있다.

## D12) 몬스터 약점 / 저항 데이터 표 작성 기준 (개정 LOCK)

- 약점 / 저항 표에는 아래 항목만 사용한다.
  - 물리 타입: 참격 / 관통 / 타격
  - 기본 원소 속성: 물 / 불 / 바람
  - 상위 원소 속성: 빛 / 어둠
  - 최상위 속성: 초월

- 물리 스킬은 기본적으로 무속성으로 취급한다.
- 별도의 속성이 부여되지 않은 물리 공격은 속성 약점 / 저항 판정을 적용하지 않는다.
- 속성이 부여된 경우에만 해당 속성의 약점 / 저항 판정을 적용한다.
- 빛 / 어둠은 상위 원소 속성이며, 룬을 통해 부여된 경우에만 관련 약점 / 저항 판정을 적용한다.
- 초월은 최상위 속성이며, 특수 룬 또는 예외적 효과를 통해서만 부여된다.
- **독은 속성이 아니라 상태이상**이므로 약점 / 저항 표에 직접 넣지 않는다.
- 출혈 / 기절 / 한기 / 감전은 삭제하며, 약점 / 저항 표에 사용하지 않는다.
- 자연 / 심연 / 공포 / 지연은 약점 / 저항 표에서 사용하지 않는다.
