## [v2.4.2] - 2026-03-05
### Added
- CORE/DATA/DESIGN/APPENDIX의 **3-레이어 문서 운영 방식**.
- **빙의 캐릭터 12명**(각 종족 2명) 이름/직업/성격/과거/목표/빙의 독백.
- 각 캐릭터 **시엔 6 + 베라 1** 스킬 최종안(솔로 전용).
- 라그니스 흔적 던전 6개 고유명:
  - Crownfall Vault / Rootglass Hollow / Anvilwound Depths / Bloodecho Pit / Emberseal Crucible / Nullscript Laboratory
- 던전 레이아웃 규칙: **A-B-C(제단)-D(보스)**, 전장 크기 가이드(6×5×2 등).
- 카메라 가이드: **HD-2D 변형(3D 디오라마 + 도트 캐릭터), 탑다운 금지** 수치.
- 부록(APPENDIX): 저항/약점 표, 운용 팁/상성, 제단 구체 위치 콘셉트, 카메라 UX 디테일.

### Changed
- (LOCK) **총기/화약 무기 제거** → 중세 무기 + 마법/룬 투사체로 대체.
- (LOCK) 게임 플레이 전제: **솔로 플레이** 강화 → 순수 서포터 직업/스킬 금지.
- 저장 시스템:
  - 거점: 기록대(무료)
  - 던전: 각인의 제단(던전당 1개, C구역) + **기록석 1개 소모**
  - 기록석 수급: 파편 던전 보스 확정 1개 + 필드 엘리트 낮은 확률 + 드워프 합성(조각 3→1)

### Fixed
- 용족 빙의 혼선 정리: **용족 본체 빙의 X, 용인족 빙의 O**로 정리.

### Removed
- (LOCK) 총기/탄환/화약 관련 모든 설정 요소.
