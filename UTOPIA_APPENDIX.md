### A1.1) 상성 및 저항 대미지 보정 산식 (LOCK)

- **약점 (Weakness)**: 최종 대미지 150% 적용. 타격 시 대상의 다음 턴 대기열(Timeline) 속도 10% 지연.
- **일반 (Normal)**: 최종 대미지 100% 적용.
- **저항: 경미 (Minor Resist)**: 최종 대미지 75% 적용.
- **저항: 강 (Strong Resist)**: 최종 대미지 50% 적용. 해당 속성 관련 상태이상(화상, 빙결, 중독 등) 완전 면역.

#### A1.x) 전투 용어 정리 — 사거리 / 범위 / 대상 (LOCK)

- **사거리(Range)**:
  - 스킬이나 공격이 도달할 수 있는 최대 거리.
  - 사용자를 기준으로 몇 칸 떨어진 타일까지 지정 가능한지를 뜻한다.
  - 사거리가 3이면 사용자로부터 3칸 이내의 타일을 지정할 수 있다.

- **범위(Area)**:
  - 지정한 지점을 기준으로 실제 효과가 적용되는 칸의 모양과 크기.
  - 예: 단일, 직선 3칸, 3x3, 반경 2칸.

- **대상(Target)**:
  - 범위 안에서 실제 효과를 받는 단위.
  - 단일 대상이면 적 1체만 적용된다.
  - 범위 대상이면 범위 안의 모든 적 또는 조건을 만족하는 대상에게 적용된다.

- **단일 대상 스킬 표기 규칙**:
  - 단일 공격은 `사거리 X, 대상 적 1체` 형식으로 표기한다.
  - 단일 공격에 `1칸` 표현은 사용하지 않는다.

- **범위 공격 스킬 표기 규칙**:
  - 범위 공격은 `사거리 X, 범위 Y` 형식으로 표기한다.
  - 예:
    - 사거리 3, 범위 3x3
    - 사거리 4, 범위 직선 3칸

- **유토피아 전장 기준 해석 주의**:
  - 현재 전장 크기가 5x4 ~ 7x7 수준이므로, 사거리 4는 맵 전체가 아니라도 체감상 매우 넓게 느껴질 수 있다.
  - 따라서 단일 결전기 성격의 스킬은 사거리 3 전후를 기본 검토값으로 삼는다.

## A2) 캐릭터 운용 팁 + 상성(툴팁/가이드용)

---
> “리롤” 유혹을 줄이려면, 특정 캐릭이 ‘필수’가 되지 않게 **상성은 있으나 압도는 없게** 유지.

- **Caldus**: 함정→끌기→후방 킬각→이탈. / 강점: 느린 근접·후방 취약 / 약점: Blunt 난사·번개 추적
- **Lysa**: 장판으로 동선 통제→폭발로 지형 변경. / 강점: 군집·엄폐 의존 / 약점: 독·심연 지속·순간이동
- **Eirwen**: 핀샷 고정→독 누적→2연사 마무리. / 강점: 기동 낮은 중대형 / 약점: 화염·돌진으로 거리 붕괴
- **Maelor**: 흡수 표식으로 버티며 저주로 저항을 깎아 장기전 승리. / 강점: 단일 보스 / 약점: 화염 폭발·원거리 폭딜
- **Korin**: 가드로 버티고 브레이크를 ‘부숴’ 전투를 짧게 끝냄. / 강점: 방어 높은 적·브레이크 보스 / 약점: 번개·공중 고기동
- **Brena**: 엄폐+노드로 안전지대 구축, 룬 표식 폭발로 딜. / 강점: 통로형 맵·근접 러시 / 약점: 설치물 무시·결빙/번개
- **Mokhra**: 맞으며 스택→돌진 제압→피값 폭딜(리스크-보상). / 강점: 다수전 / 약점: 냉기 봉쇄·관통 창병
- **Zugrim**: 표식→끌기/그물로 도주 삭제→점프 추격. / 강점: 회피 높은 민첩형 / 약점: 장판 화염·다수 원거리
- **Vaelis**: 암영으로 각 훔치고 연쇄 처형, 끊기면 장막으로 이탈. / 강점: 후열 마법사·저체력 정예 / 약점: Blunt 강타 보스·가드/반격 특화
- **Rhexar**: 패턴 읽기→약점 고정→베라로 ‘증명’(킬각). / 강점: 약점 뚜렷한 보스 / 약점: 돌진으로 캐스팅 끊김·지속 화염
- **Nyra**: 2층 점프+넉백으로 전장을 낙사 게임으로. / 강점: 낙사/벽꽝 맵·대형 단일 / 약점: 냉기 봉쇄·원거리 포화
- **Rhyden**: 점화 스택 깔고 순간이동으로 안전 확보→폭발로 마무리. / 강점: 기믹 많은 전투·중갑 단일 / 약점: 냉기(룬 흐름 차단)·Blunt 연타

## A3) 저장 아이템(기록석) 규칙 디테일(확정)

---
- 던전 제단에서 기록하려면 **기록석 1개 소모**.
- 거점 기록대는 무료(기본 저장).
- 수급(불합리 방지):
  - 각 라그니스 잔핵 던전 보스: **기록석 1개 확정**
  - 필드 엘리트: **낮은 확률 드롭**
  - 합성(드워프 공방): **조각 3개 → 기록석 1개**(완제품 제작/상점 무한 구매 금지)

## A6) 카메라/조작 UX 추가 디테일(선택)

---
- 탐험 카메라 요 회전: **±45° 스텝 회전** 허용, 회전 시간 **0.18~0.25s**.
- 자동 회전은 최소화하고, 전투 조준 중 자동 회전은 **±10° 제한**.
- 가림 처리(권장): 캐릭터-카메라 사이 오브젝트는 거리/각도 조건 충족 시 **반투명화**.
